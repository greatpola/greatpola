## 🚀 How to experience Agent 8

Agent 8의 코어 소스 코드는 비공개(Closed-source)로 유지되고 있으나, 깃허브 공간을 활용하여 클라이언트 앱 및 최신 기능을 배포하고 있습니다. 

### 1️⃣ 🚀 Agent 8 Updates & Release Notes
* 저희의 가장 최신 업데이트 사항, 에이전트 기능 개선 및 패치 노트는 저장소의 **[Releases](../../releases)** 탭에서 확인할 수 있습니다. 에이전트들의 지능이 어떻게 진화하고 있는지 지켜봐 주세요.

### 2️⃣ 💻 Mac OS App Installation
* Agent 8을 데스크탑 환경에서 더욱 간편하게 사용하기 위한 **Mac OS 전용 앱(dmg)**을 만나보세요. 최신 버전 앱은 [Releases](../../releases) 페이지에서 다운로드하여 설치하실 수 있습니다.

### 3️⃣ 🌐 Chrome Browser Extension
* 웹 브라우저의 컨텍스트를 즉시 인식하고 에이전트가 행동할 수 있게 도와주는 브라우저 익스텐션입니다. 익스텐션 폴더를 다운로드하여 크롬 확장 프로그램 관리자에서 등록 후 사용할 수 있습니다.

## 💡 About Agent 8 System

* **PentAGI Delegation Engine**: 단순히 태스크를 수행하는 챗봇이 아닌 자율적으로 검토하고 작업을 위임합니다.
* **Nanjung Ilgi CRM**: "본 것만 기록한다"는 이순신 장군의 증거 기반 상태 추적 (Evidence-based Logging) 방식 원칙.
* **Make It Mindset**: 확실한 실패 증거를 찾기 전까지 절대 포기하지 않는 집요함.
* **Real-time SSE Discussion**: 파트너 간 실시간 토론 및 객관적 합의 루프 구동.
