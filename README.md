## 🚀 How to Experience Agent 8

현재 Agent 8의 코어 엔진 소스 코드는 시스템 보안 및 지식 자산 보호를 위해 비공개(Closed-Source) 정책을 유지하고 있습니다. 대신 본 GitHub 공간을 통해 Agent 8과 원활하게 소통할 수 있는 안내와 클라이언트 앱 배포 공간으로 활용하고 있습니다.

* **🚀 Agent 8 Updates & Release Notes**: 에이전트 시스템 로직의 고도화 업데이트 및 패치노트는 향후 저장소의 릴리즈(Releases) 탭을 통해 문서화될 예정입니다.
* **💻 Mac OS App Installation**: 데스크탑 런타임 환경에서 시스템 리소스를 활용하며 Agent 8을 네이티브하게 사용하기 위한 Mac 전용 설치 파일(dmg)이 릴리즈를 통해 배포될 예정입니다.
* **🌐 Chrome Browser Extension**: 브라우저 서핑 중 실시간 컨텍스트를 즉시 인식하고, Agent 8이 작업을 수행할 수 있도록 돕는 확장 프로그램이 제공될 예정입니다.

## 💡 Philosophy & Rules

* **PentAGI Delegation Engine**: 수동적인 어시스턴트가 아닙니다. 작업을 스스로 분할하고 적합한 파트너에게 자율적으로 권한을 넘겨주며 해결해 나갑니다.
* **Nanjung Ilgi CRM**: "눈으로 보고 검증한 것만 남긴다"는 확고한 철학을 바탕으로, 할루시네이션(환각) 없는 증거 기반 상태 추적(Evidence-based Logging)만을 유지합니다.
* **Make It Mindset**: 완벽한 실패의 증거를 찾기 전까지 안 된다는 말을 거부하고, 대안을 찾아 집요하게 물고 늘어집니다.
* **Real-time SSE Discussion**: 독단적으로 결론을 내리지 않습니다. 이해상충이 존재하는 파트너들이 모여 치열하게 토론하며 객관적인 합의점을 끌어냅니다.
